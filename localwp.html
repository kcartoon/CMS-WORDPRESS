Local-wp

Téléchargement

Pour commencer vous allez installer le logiciel local-wp qui est l’outil nécessaire pour installer un serveur local dédié à WordPress et préparer son environnement de travail :

Aller sur le site https://localwp.com/ et le télécharger

  • Choisir download for free
  • Choisir votre plateforme

Echo en retour

Vous avez le choix de ne pas vous inscrire mais si vous voulez pendant votre phase de test qu’il soit visible par d’autres personnes, vous devez impérativement vous inscrire. Nous verrons comment utiliser cette option plus tard.

Installation et configuration

  • Lancer le logiciel Local-wp
  • Cliquer sur logo en haut à gauche => Préférences
  • Suivre les étapes suivantes

  • Sur l’onglet New site defaults, mettre votre adresse e-mail

  • Cliquer sur Browse, une fenêtre s’ouvre, vous choisissez le lieu de stockage de votre projet, si vous devez l’utiliser dans différents lieux il est souhaitable de le stocker sur votre drive en ayant  préalablement créer en amont un dossier sur celui-ci.
  • Fermer la fenêtre avec la croix, vous reviendrez sur la page principale.
